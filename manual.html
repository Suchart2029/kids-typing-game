⚙️ 2. วิธีการทำและขั้นตอนการผลิต

1. Wireframing

จัดระเบียบสีคีย์บอร์ดจำลองให้โดดเด่นสะดุดตาเด็ก

2. Input Coding

เขียนโปรแกรมดักจับตำแหน่งรหัสปุ่มกายภาพไม่ค้างไทย

3. Audio Syn

ออกแบบระบบคลื่นเสียงแบบนุ่มนวลรักษาคลื่นสมอง

4. Leaderboard

จัดโครงสร้างตารางเซฟคะแนนภายในเครื่องแยกบุคคล

🎮 3. วิธีการใช้งานสื่อ (How to Play)

A

กรอกชื่อเล่นและคัดสรรการ์ตูนสัตว์ผู้พิทักษ์เพื่อสร้างโปรไฟล์

B

เลือกโหมดการพิมพ์: จิ้มลูกโป่งลอยฟ้า หรือ สวนสัตว์สะกดคำ

C

กดปุ่มสีเหลืองเรืองแสงกระพริบบอกใบ้ตามคีย์บอร์ดจำลองบนจอ

D

ตรวจสอบอันดับถ้วยรางวัลคนพิมพ์เก่ง 1-3 ในหอเกียรติยศ
